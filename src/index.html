RISC-V SoC Simulator

                                    
SoC Simulator là một công cụ mô phỏng hoạt động của một Hệ thống trên Chip (SoC)
                                        được thiết kế và phát triển bởi nhóm tác giả Nguyễn Xuân Lộc & Dương Hiển Gia
                                        Khang. Mục đích của Project là hỗ trợ những người mới bắt đầu tìm hiểu về thiết
                                        kế SoC.

                                

                                

                                    RV32I
                                    RV32M
                                    RV32F
                                    MMIO
                                    Cache
                                

                            

                            

                                Mục tiêu sử dụng: viết assembly trong Editor, bấm Assemble để sinh mã
                                máy, Run/Step để thực thi, rồi quan sát Registers, Memory, Cache, I/O và Systems Log
                                Console.
                            

                            

                                
                                    

                                        Khu vực
                                        Dùng để làm gì
                                        Mẹo thao tác
                                    

                                
                                
                                    

                                        Editor
                                        Soạn RISC-V assembly, đặt breakpoint ở gutter, xem binary output.
                                        Ưu tiên khai báo .text, .globl _start, nhãn
                                            _start:.
                                        
                                    

                                    

                                        Processor
                                        Không gian dành cho datapath/processor visualization.
                                        Dùng Registers và Instruction Memory để debug tạm thời.
                                    

                                    

                                        Cache
                                        Xem L1I, L1D, L2 cache: set, way, valid, dirty, tag, hit/miss.
                                        Dùng nút Cache: ON/OFF để so sánh hành vi bộ nhớ.
                                    

                                    

                                        Memory
                                        Xem instruction memory và data segment.
                                        Ô địa chỉ hỗ trợ dạng thập phân hoặc hex, ví dụ 0x10010000.
                                        
                                    

                                    

                                        I/O
                                        Tương tác LED Matrix, UART Console, Keyboard.
                                        I/O dùng memory-mapped registers, không phải syscall RARS truyền thống.
                                    

                                
                            

                        


                        

                            
Quy trình chạy chương trình

                            
    
                                
  1. Viết code: đặt dữ liệu trong .data, đặt lệnh trong
                                    .text. Nếu có nhãn _start, simulator sẽ ưu tiên chạy từ
                                    nhãn này.
                                
    2. 
                                
  2. Assemble: bấm Assemble để kiểm tra cú pháp, expand
                                    pseudo-instruction và nạp chương trình vào CPU/memory.
    3. 
                                
  3. Sửa lỗi: nếu có lỗi, đọc dòng lỗi trong Console / Binary Output.
                                    Lỗi phổ biến là sai số toán hạng, label chưa khai báo, immediate vượt range.
    4. 
                                
  4. Step: chạy từng instruction để xem PC, register, memory, cache thay
                                    đổi theo từng chu kỳ.
    5. 
                                
  5. Run: chạy liên tục. Nếu đã đặt breakpoint, Run sẽ dừng khi PC tới
                                    instruction tương ứng.
    6. 
                                
  6. Quan sát: Registers tô vàng khi giá trị đổi; Memory cho biết mã
                                    máy/data; I/O hiển thị UART/LED/keyboard; Systems Log Console ghi log bus, CPU, DMA,
                                    cache.
    7. 
                            

                            

                                
                                    

                                        Nút
                                        Chức năng
                                    

                                
                                
                                    

                                        Assemble
                                        Biên dịch assembly, reset CPU/memory cần thiết và load chương trình.
                                    

                                    

                                        Reset
                                        Reset trạng thái CPU/simulator nhưng giữ code trong editor.
                                    

                                    

                                        Run
                                        Thực thi liên tục tới khi program exit, gặp lỗi, vượt max cycle hoặc hit
                                            breakpoint.
                                    

                                    

                                        Step
                                        Thực thi đúng một bước simulator, hữu ích khi kiểm tra PC/register.
                                    

                                    

                                        Speed
                                        Điều chỉnh số chu kỳ mỗi frame khi Run.
                                    

                                    

                                        IPS
                                        Ước lượng số instruction/chu kỳ đang chạy mỗi giây.
                                    

                                    

                                        Cache: ON/OFF
                                        Bật/tắt cache rồi quan sát khác biệt trong Cache view và log.
                                    

                                
                            

                        


                        

                            
Cú pháp assembly & operand key

                            

                                

                                    
Quy ước dòng lệnh

                                    
    
                                        
  • Comment bắt đầu bằng #.
    • 
                                        
  • Label kết thúc bằng dấu hai chấm: loop:, done:.
                                        
    • 
                                        
  • Immediate hỗ trợ decimal và hex: 10, -4,
                                            0x10010000.
                                        
    • 
                                        
  • Load/store dùng dạng offset(base), ví dụ
                                            lw t0, 8(sp).
                                        
    • 
                                        
  • Branch/jump tới label dùng PC-relative offset do assembler tự tính.
    • 
                                        
  • Không đặt instruction trong .data; chuyển về .text
                                            trước khi viết lệnh.

Operand key

                                    

                                        
                                            

                                                rd
                                                Destination register, nơi ghi kết quả.
                                            

                                            

                                                rs1, rs2
                                                Source registers.
                                            

                                            

                                                imm12
                                                Immediate 12-bit có dấu, thường trong I/S-type.
                                            

                                            

                                                label
                                                Nhãn trong chương trình, dùng cho branch/jump/la/call.
                                            

                                            

                                                offset(base)
                                                Địa chỉ hiệu dụng = base + offset.
                                            

                                            

                                                frd, frs1
                                                Floating-point registers f0..f31 hoặc ABI như
                                                    fa0.

    ecall

                                        Nhóm
                                        Lệnh
                                        Cú pháp
                                        Ý nghĩa
                                    

                                
                                
                                    

                                        R-type
                                        add sub
                                        rd, rs1, rs2
                                        Cộng/trừ hai thanh ghi.
                                    

                                    

                                        R-type
                                        sll srl sra
                                        rd, rs1, rs2
                                        Dịch trái logic, dịch phải logic, dịch phải số học; shift amount lấy 5 bit
                                            thấp của rs2.
                                    

                                    

                                        R-type
                                        slt sltu
                                        rd, rs1, rs2
                                        So sánh signed/unsigned, ghi 1 nếu rs1 nhỏ hơn rs2.
                                    

                                    

                                        R-type
                                        xor or and
                                        rd, rs1, rs2
                                        Toán tử bitwise.
                                    

                                    

                                        I-type
                                        addi slti sltiu xori ori andi
                                        rd, rs1, imm12
                                        Phép toán với immediate 12-bit có dấu.
                                    

                                    

                                        I-shift
                                        slli srli srai
                                        rd, rs1, shamt
                                        Dịch bit với shift amount 0..31.
                                    

                                    

                                        Load
                                        lb lh lw lbu lhu
                                        rd, offset(rs1)
                                        Đọc byte/half/word từ memory; lbu/lhu zero-extend.
                                    

                                    

                                        Store
                                        sb sh sw
                                        rs2, offset(rs1)
                                        Ghi byte/half/word từ rs2 xuống memory hoặc MMIO.
                                    

                                    

                                        Branch
                                        beq bne blt bge bltu bgeu
                                        rs1, rs2, label
                                        Nhảy có điều kiện; signed hoặc unsigned tùy lệnh.
                                    

                                    

                                        Jump
                                        jal
                                        rd, label
                                        Ghi pc+4 vào rd rồi nhảy tới label.
                                    

                                    

                                        Jump
                                        jalr
                                        rd, rs1, imm hoặc rd, imm(rs1)
                                        Nhảy tới (rs1+imm)&~1, thường dùng cho return/function pointer.
                                        
                                    

                                    

                                        Upper
                                        lui auipc
                                        rd, imm20
                                        Nạp immediate cao hoặc cộng offset cao với PC.
                                    

                                    

                                        System
                                        ecall ebreak fence
                                        ecall, ebreak, fence pred, succ
                                        System call, breakpoint exception, memory ordering hint.

Pseudo instructions

                            

                                
                                    

                                        Pseudo
                                        Expand gần đúng
                                        Ghi chú
                                    

                                
                                
                                    

                                        nop
                                        addi x0, x0, 0
                                        Không làm gì.
                                    

                                    

                                        li rd, imm
                                        addi hoặc lui + addi
                                        Assembler tự chọn 1 hoặc 2 instruction theo độ lớn immediate.
                                    

                                    

                                        la rd, label
                                        auipc + addi
                                        Nạp địa chỉ label đúng theo PC-relative relocation đơn giản.
                                    

                                    

                                        mv rd, rs
                                        addi rd, rs, 0
                                        Copy register.
                                    

                                    

                                        j label
                                        jal x0, label
                                        Nhảy không lưu return address.
                                    

                                    

                                        jr rs
                                        jalr x0, rs, 0
                                        Nhảy tới địa chỉ trong register.
                                    

                                    

                                        ret
                                        jalr x0, ra, 0
                                        Return từ function.
                                    

                                    

                                        call label
                                        auipc ra + jalr ra
                                        Gọi subroutine xa hơn range của jal.
                                    

                                    

                                        beqz rs, label
                                        beq rs, x0, label
                                        Branch nếu register bằng 0.
                                    

                                    

                                        bnez rs, label
                                        bne rs, x0, label
                                        Branch nếu register khác 0.
                                    

                                    

                                        fmv.s, fabs.s, fneg.s
                                        fsgnj.s variants
                                        Copy, absolute, negate float bằng sign injection.

ECALL / Syscall

                            

                                Đặt syscall ID trong a7, đặt arguments trong a0,
                                a1, a2, sau đó gọi ecall. Trong trình duyệt, các
                                syscall in/exit hiển thị trong Console / Binary Output; UART MMIO vẫn dùng cho I/O thiết bị.
                            

                            

                                
                                    

                                        a7
                                        Tên
                                        Input
                                        Output / tác dụng
                                    

                                
                                
                                    

                                        1
                                        PrintInt
                                        a0 = integer
                                        In số nguyên.
                                    

                                    

                                        4
                                        PrintString
                                        a0 = địa chỉ chuỗi null-terminated
                                        In chuỗi tới khi gặp byte 0.
                                    

                                    

                                        64
                                        Write stdout
                                        a0=1, a1=buffer, a2=count
                                        Ghi count byte; trả số byte đã ghi trong a0.
                                    

                                    

                                        93
                                        Exit
                                        a0 = exit code
                                        Dừng chương trình.
                                    

                                    

                                        0
                                        Halt/no-op
                                        Không cần
                                        Dừng CPU theo behavior hiện tại.
                                    

                                
                            

                        


                        

                            
Memory map

                            

                                
                                    

                                        Range / base
                                        Tên
                                        Ghi chú lập trình
                                    

                                
                                
                                    

                                        0x00400000
                                        Text segment
                                        Mặc định cho instruction sau .text.
                                    

                                    

                                        0x10010000
                                        Data segment
                                        Mặc định cho dữ liệu sau .data.
                                    

                                    

                                        0x10000000-0x10000013
                                        UART
                                        TX/RX/status/control/baud divisor.
                                    

                                    

                                        0xFF000000-0xFF000FFF
                                        LED Matrix
                                        32x32 pixels, mỗi pixel 1 word 0x00RRGGBB.
                                    

                                    

                                        0xFF100000-0xFF100013
                                        Mouse
                                        X, Y, button bitmap, status, control.
                                    

                                    

                                        0xFFFF0000-0xFFFF0007
                                        Keyboard
                                        Control bit0 = ready; Data = ASCII char.
                                    

                                    

                                        0xFFED0000-0xFFED0007
                                        DMA registers
                                        CTRL và DESC FIFO.
                                    

                                
                            

                        


                        

                            
I/O peripherals

                            

                                

                                    
UART 0x10000000

                                    

                                        
                                            

                                                +0x00
                                                TX write-only: ghi ASCII byte để gửi ra UART Console.
                                            

                                            

                                                +0x04
                                                RX read-only: đọc byte người dùng nhập.
                                            

                                            

                                                +0x08
                                                Status: bit0 TX ready, bit1 RX available, bit2/3 interrupt enable.
                                                
                                            

                                            

                                                +0x0C
                                                Control: bit0 TX interrupt enable, bit1 RX interrupt enable.
                                            

                                            

                                                +0x10
                                                Baud divisor; baud = 48MHz / (16 x divisor).
                                            

                                        
                                    

                                

                                

                                    
LED Matrix 0xFF000000

                                    
Mỗi pixel là một word. Offset pixel = ((row * 32) + col) * 4. Màu
                                        dùng dạng 0x00RRGGBB.

                                    
Keyboard 0xFFFF0000

                                    
Đọc control +0x00, nếu bit0 = 1 thì đọc data +0x04 để
                                        lấy ASCII và pop khỏi buffer.

                                    
Mouse 0xFF100000

                                    
Đọc X/Y/button/status ở +0x00/+0x04/+0x08/+0x0C. Ghi
                                        0x3 vào status để clear move/click bits.
                                    

                                

                            

                            
DMA quick reference

                            

                                
                                    

                                        CTRL +0x00
                                        bit0 EN, bit1 START, bit27 FIFO empty, bit28 FIFO full, bit29 error, bit30
                                            done, bit31 busy.
                                    

                                    

                                        DESC +0x04
                                        Write-only FIFO: ghi 3 word theo thứ tự source, destination, config.
                                    

                                    

                                        config
                                        bits 0..23 = length/elements, bit27 byte swap, bits 28..29 source mode, bits
                                            30..31 destination mode.
                                    

                                
                            

                        


                        

                            
Cache, MMU và bus

                            
    
                                
  • Đường CPU chính đi qua
                                    RISC-V Core -> MMU -> Cache -> TileLink-UH -> Main Memory.
                                
    • 
                                
  • MMIO như UART/LED/Keyboard/Mouse đi qua TileLink-UL và thường không cache.
    • 
                                
  • Cache view có 3 tab: L1I cho instruction fetch, L1D cho data access, L2 shared
                                    cache.
    • 
                                
  • Khi Cache ON, bảng cache hiển thị set/way/tag/valid/dirty và thống kê
                                    read/write/hit/miss.
    • 
                                
  • MMU hiện có page table/TLB nhỏ ở mức simulator; log MMU giúp thấy translate,
                                    hit/miss và permission fault.
    • 
                                
  • Dùng Systems Log Console để thấy request/response CPU, MMU, cache, TileLink, DMA
                                    theo trình tự thời gian.
    • 
                            

                        


                        

                            
Debugging checklist

                            

                                
                                    

                                        Triệu chứng
                                        Nguyên nhân hay gặp
                                        Cách kiểm tra
                                    

                                
                                
                                    

                                        Unknown mnemonic/directive
                                        Sai tên lệnh hoặc directive chưa hỗ trợ.
                                        Tra bảng Instruction set/Directives; chú ý dấu chấm trong lệnh float như
                                            fadd.s.
                                        
                                    

                                    

                                        Immediate out of range
                                        addi/lw/sw chỉ có immediate 12-bit; branch/jump có range riêng.
                                        
                                        Dùng li để nạp hằng lớn hoặc tách lui/addi.
                                    

                                    

                                        Không nhảy đúng label
                                        Label đặt trong section sai hoặc branch offset không align.
                                        Assemble xong xem Instruction Memory và cột Source.
                                    

                                    

                                        Register không đổi
                                        Ghi vào x0, hoặc lệnh chưa chạy do PC/branch.
                                        Step từng lệnh, quan sát PC và dòng được highlight.
                                    

                                    

                                        UART không in ký tự
                                        TX chưa ready hoặc địa chỉ sai.
                                        Poll bit0 ở UART_STATUS trước khi ghi TX.
                                    

                                    

                                        LED không sáng
                                        Tính sai offset pixel hoặc màu bằng 0.
                                        Offset phải nhân 4; màu nên dạng 0x00RRGGBB.
                                    

                                    

                                        Run quá lâu
                                        Vòng lặp vô hạn không có exit/breakpoint.
                                        Đặt breakpoint hoặc thêm li a7,93; ecall.
                                    

                                
                            

                        


                        

                            
Ví dụ mẫu

                            

                                

                                    

                                        
UART: in hai ký tự

                                        
Ghi ASCII vào TX, poll status trước khi gửi byte tiếp theo.

                                    

                                    
                                

                                

                                    

                                        
LED Matrix: tô gradient

                                        
Duyệt 32x32 pixel và ghi word màu 0x00RRGGBB.

                                    

                                    
                                

                                

                                    

                                        
Syscall: in chuỗi rồi exit

                                        
Dùng a7=4 để print string và a7=93 để exit.
