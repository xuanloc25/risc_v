# RISC-V Simulator & Assembler

Đây là dự án mô phỏng Hệ thống trên Chip (System-on-Chip Simulator) và trình biên dịch hợp ngữ (Assembler) dành cho kiến trúc RISC-V. Ứng dụng cung cấp giao diện trực quan để viết code, biên dịch và chạy mô phỏng các tập lệnh RISC-V.
